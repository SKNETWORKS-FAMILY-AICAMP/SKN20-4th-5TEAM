# 계획: 네이버 API 교체 및 코드 보존

이 계획은 `langgraph_agent.py`에서 기존 카카오 API를 네이버 지역 검색 API로 교체하고, 기존 코드를 주석으로 보전하는 단계를 설명합니다.

## 제안된 변경 사항

### [백엔드 (FastAPI)]

#### [수정] [langgraph_agent.py](file:///d:/SKN20-4th-5TEAM/backend/app/services/langgraph_agent.py)
- **기존 코드 보존**: 현재 일부 삭제된 카카오 API 관련 로직을 복구하여 주석 처리합니다.
- **네이버 API 연동**:
    - `requests` 라이브러리를 사용하여 네이버 지역 검색 API(`search/local.json`)를 연동합니다.
    - JSON 응답 파싱 및 예외 처리를 추가합니다.
- **날짜 기록**: 모든 수정 사항에 `2026-01-06` 주석을 추가합니다.

## 검증 계획

### 자동 테스트
- 백엔드 서버 로그를 통해 네이버 API 호출 성공 여부 및 검색 결과 확인.

### 수동 검증
- 챗봇에 특정 장소(예: "서울역")를 입력하여 대피소 검색 결과가 정상적으로 나오는지 확인.
