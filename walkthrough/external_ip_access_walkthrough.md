## 2. 서버 실행 상태 (0.0.0.0 바인딩)

| 구성 요소 | 명령업 | 호스트/포트 |
| :--- | :--- | :--- |
| **Backend** | `uvicorn backend.app.main:app` | `0.0.0.0:8001` |
| **Frontend** | `python manage.py runserver` | `0.0.0.0:8000` |

> [!NOTE]
> `0.0.0.0` 바인딩을 통해 내부 로컬 접속(`localhost`)은 물론, 서버의 공인/사설 IP를 통한 모든 외부 접속 수신이 가능해졌습니다.

## 3. 향후 조치 권장
외부에서 프론트엔드 접속 시 백엔드 API와 원활하게 통신하려면, `.env` 파일의 `FASTAPI_HOST` 값을 실제 외부 접근 가능한 IP주소로 설정하시기 바랍니다.
